# PVC-vloeren op terrassen Gallery Images

This folder contains images for the "PVC-vloeren op terrassen" gallery.

## Instructions for adding images:

1. Add your images to this folder with descriptive names (e.g., pvc1.jpg, pvc2.jpg, etc.)
2. Recommended image sizes: 1200x800px or similar aspect ratio
3. Supported formats: .jpg, .jpeg, .png, .webp

## To update the gallery:

1. Add your images to this folder
2. Update the 'general' category in src/components/GalleryModal.tsx
3. Add entries with:
   - imageUrl: "/images/pvc-vloeren-terrassen/your-image.jpg"
   - title: "PVC-vloeren op terrassen"
   - description: "Duurzame PVC-vloeren aangelegd op terrassen, stijlvol en onderhoudsvriendelijk."

## Current gallery category:
This gallery uses the 'general' category in GalleryModal.tsx.
